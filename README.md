# 🔐 Cybersecurity Port Scanner

A Python-based port scanner that identifies open ports and associated services on a target machine.

## Features
- Scans ports 1–1024
- Detects open ports
- Identifies common services (HTTP, SSH, etc.)

## Technologies
- Python
- Socket Programming

## Purpose
This project demonstrates foundational cybersecurity and networking concepts,
including port scanning and service detection.

## How to Run
```bash
python scanner.py
